# XCMania中国市场版
 
## 手机需求

- [x] 需要Android 9.0或更高手机（相当于2018年购买的手机）
- [x] 需要配备有气压传感器的手机 / 或者连接蓝牙高度表使用。诸如中国的“蜻蜓”或瑞士的XCtracer等等。

由于XCMania在飞行中计算量与3D渲染比9年前诞生的XC高一个量级。你需要一部上市日期2018年或之后的当代手机。
